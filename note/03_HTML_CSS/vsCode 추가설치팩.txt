System Installer를 다운받아 추가설치한다
유저 버전은 설치하고 있는 현재 윈도우 사용자 계정만 사용가능.
시스템버전은 해당 컴퓨터에 모든 사람이 사용 가능

korean (한글팩설치 후 resart해줘야 함)
Prettier - Code formatter(prettier로 검색 ; 코드 포맷팅)
open in browser (브라우저로 볼 수 있는 프로그램 Alt+B)
indent-rainbow(indent - 들여쓰기 레벨을 보기 쉽게 색으로 구별)
Auto Rename Tag(rename - 여는 태그를 바꾸면 닫는 태그도 자동 변경)
Live Server(Alt + L O ) - Open with Live Server
