자바(JDK) 설치
내PC 오른쪽 마우스 – 속성 - 고급시스템설정 - 환경변수
시스템변수 새로만들기-변수이름:JAVA_HOME  / 변수값 : JDK설치된 폴더
Path에  ; %JAVA_HOME%\bin; 추가


cmd 실행 후 자바 버전 확인 (java - version)
자바 버전에 맞는 이클립스 설치

int(데이터타입)    i(변수명)   =(할당연산자) 10(변수내용물) ; // 선언과 할당
(주머니 성질)    (주머니이름)                    (주머니내용물)

int j; // 선언
j = 10; // 값할당

[]배열은 변수의 모음집
배열의 단점은 확보된 공간 외적으로 추가를 할 수 없다(수정 불가)
JAVA Collection은 배열의 단점이 보완되므로 더 자주 씀

다차원배열은 자바에서 잘 다루지않음 (거의 파이썬에서만)

트리셋컬렉션에 난수 구현
